Используемые формулы:

  • Периметр: P = a + b + c
  • Полупериметр: p = P/2
  • Площадь (по формуле Герона): S = √[p(p-a)(p-b)(p-c)]
  • Высота к стороне a: hₐ = 2S/a
  • Медиана к стороне a: mₐ = 0.5√(2b²+2c²-a²)
  • Биссектриса к стороне a: lₐ = (2√[bcp(p-a)])/(b+c)
